## Installation
1. `git clone https://github.com/JustMaffie/RandomBot`
1. `cd RandomBot`
1. `python3.5 -m pip install -r requirements.txt`
1. `cp config.example.json config.json`
1. Edit `config.json`
1. `sudo sh ./run.sh`
